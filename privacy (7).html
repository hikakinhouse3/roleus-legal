プライバシーポリシー

RoleUs

最終更新日：2026年7月5日

本プライバシーポリシーは、RoleUs（以下「本アプリ」）における利用者の情報の取り扱いについて定めるものです。本アプリは、利用者のプライバシーを尊重し、必要最小限の情報のみを扱うことを基本方針としています。

1. アカウント登録について

本アプリはアカウント登録やログインを必要としません。氏名・メールアドレス・電話番号などの個人を特定する情報を、利用者から取得することはありません。

2. 端末内に保存される情報

本アプリで作成・入力される以下の情報は、原則として利用者の端末内にのみ保存され、運営者を含む外部に自動的に送信されることはありません。

  • 役割診断の結果
  • 登録したチームメンバーの情報および役割
  • プロジェクト、登録した資料（テキスト・PDF・画像）
  • 生成された文書、相談の履歴

これらの情報は、アプリを削除すると端末から消去されます。

3. AI機能の利用に伴う情報の送信

本アプリのAI機能（資料の整形、タスクの割り当て、相談など）を利用する場合に限り、利用者が入力した内容が、処理のために外部のAIサービスへ送信されます。

送信される主な内容

  • 利用者が入力・添付した資料やメッセージの内容
  • 役割分担の生成などに用いるメンバーの役割情報

これらの情報は、運営者が管理する中継サーバー（Cloudflare Workers）を経由して、AIサービス提供者である Anthropic 社の API に送信され、応答の生成にのみ利用されます。送信内容は、Anthropic 社のポリシーに基づき、原則としてAIモデルの学習には使用されません。詳細は Anthropic 社のプライバシー方針をご確認ください。

AI機能を利用しない場合、これらの情報が送信されることはありません。機微な情報や第三者の個人情報を入力する際は、利用者ご自身の判断と責任においてご利用ください。

4. 課金情報について

本アプリの有料プラン（サブスクリプション）の購入は、Apple の App Store を通じて行われます。決済に関する情報は Apple により処理され、運営者がクレジットカード番号などの決済情報を取得・保管することはありません。

5. 第三者サービス

本アプリは、機能の提供のために以下の第三者サービスを利用しています。各サービスの情報の取り扱いについては、それぞれの提供者の定めに従います。

  • Anthropic（AIによる文章生成・相談機能）
  • Cloudflare（通信の中継）
  • Apple（アプリの配信および課金処理）

6. 情報の第三者提供

運営者は、法令に基づく場合を除き、利用者の情報を第三者に販売・提供することはありません。

7. 本ポリシーの変更

本ポリシーは、必要に応じて変更されることがあります。重要な変更を行う場合は、本アプリまたは本ページ上で告知します。
